_Front-end project using Random User Generator API. You can visualize, view the detail of each contact, create a new one, edit, delete, and save it to favourites._


### To launch Contacts App ðŸš€

Get a clone from the repository: 
```
git clone https://github.com/miguelgz95/contacts-app.git
```

In local, access the project and type in the terminal: 
```
npm install
```
IMPORTANT!: Once the application is started, you will need to authenticate with credentials, but to make NexAuth work, you need to create a .env.local file in the root of the project and insert code generated by entering the following command in your terminal:

```
openssl rand -base64 32
```
